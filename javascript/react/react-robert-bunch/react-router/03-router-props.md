# 03 `Router` props

Pour que le `Router` passe ses `props` à un composant, celui-ci doit être appelé par une `<Route>`.

```jsx
<Router>
    <Route path="/" component={NavBar} />
</Router>
```

IL a alors accès à des *utilitaires* dans ses `props`.

`Navbar.js`

```jsx
function NavBar(props) {

    console.log(props);
    // ...
}
```
